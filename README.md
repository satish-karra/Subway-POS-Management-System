# Subway POS Management System

A C# .NET Windows Forms based Subway Point of Sale (POS) application.

## Features
- Bread size and type selection
- Meat and vegetable customization
- Customer management (File-based storage)
- Receipt generation
- Student discount option
- Delivery option with address display

## Technologies Used
- C#
- .NET Windows Forms
- File Handling (Customer.txt)
